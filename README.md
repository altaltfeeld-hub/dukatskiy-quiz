# 🏆 Dukatskiy Quiz (Квиз Клуба Дукатов)

Modern, interactive quiz application built with React, Vite, and Socket.IO for real-time multiplayer synchronization.

## ✨ Features
- **Real-time Sync**: Host and players see the same state instantly.
- **Glassmorphism UI**: Premium, modern design with smooth animations.
- **Dynamic Backgrounds**: Mouse-reactive mesh gradients.
- **Super Game**: Special final round with timer and custom scoring.
- **Mobile Responsive**: Works perfectly on phones and desktops.

## 🛠 Tech Stack
- **Frontend**: React + Vite + Framer Motion
- **Backend**: Node.js + Express + Socket.IO
- **Styling**: Vanilla CSS (Glassmorphism)

## 🚀 Local Development
1. Clone the repository
2. Install dependencies: `npm install`
3. Run development mode: `npm run dev`
4. Access at `http://localhost:5173`

## 🌍 Production
- **Build**: `npm run build`
- **Serve**: `npm start`
- Deployment ready for **Render** or **Railway**.
